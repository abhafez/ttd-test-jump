# tdd-test-jump
test driven development test jump


## Features

Enables user to jump between file and it's test using

 ```ctrl + alt + t```

 it works with files like this

 - User.test.js => User.js
 - User.spec.js => User.js
 - user_spec.rb => user.rb
 - user_spec.rb => user.rb
 - UserSpec.ext => User.ext
